# TDD and Unit Testing
## Unittest and Pytest
### TDD
- What is it?
    - Test driven development
- How to implement?
    - Design tests and then write code to satisfy them.

### Unit testing
- What is it?
    - Testing code a small bit at a time
- What are the benefits?
    - Maintainable
    - Test a bit at a time
    
### Tools/Frameworks for Python

#### Unittest
- Unittest comes with packaged with Python

#### Pytest
- Pytest needs to be installed as a package.
- However, it is pretty and clean.
